# 📚 Book Store Management System

## 📝 Project Summary

The **Book Store Management System** is a web-based application developed using **Node.js**, **Express.js**, **MongoDB**, **EJS**, and **Multer**. It allows users to efficiently manage book records through a user-friendly interface.

### 🚀 Features

#### 📚 Add New Book

* Add book details such as:

  * 📖 Book Title
  * ✍️ Author Name
  * 🏷️ Category
  * 💰 Price
  * 📦 Quantity
  * 📝 Description
  * 🖼️ Book Cover Image
* Upload book cover images using **Multer**.

#### 👀 View All Books

* Display all books stored in the database.
* Show:

  * 🖼️ Book Image
  * 📖 Title
  * ✍️ Author
  * 🏷️ Category
  * 💰 Price
  * 📦 Quantity

#### ✏️ Update Book Details

* Edit existing book information.
* Update:

  * 📖 Title
  * 💰 Price
  * 📦 Quantity
  * 🖼️ Book Image

#### 🗑️ Delete Book

* Remove unwanted book records permanently from the database.

#### 🖼️ Image Upload Management

* Upload and store book cover images.
* Save image path in MongoDB.
* Display uploaded images on frontend pages.

### 🛠️ Technologies Used

| Technology    | Purpose             |
| ------------- | ------------------- |
| 🟢 Node.js    | Backend Runtime     |
| 🚂 Express.js | Server & Routing    |
| 🍃 MongoDB    | Database            |
| 🐹 Mongoose   | Database Modeling   |
| 🎨 EJS        | Frontend Templating |
| 📂 Multer     | Image Upload        |
| 🔄 Nodemon    | Auto Server Restart |
